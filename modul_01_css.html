01

Pengenalan CSS & Cara Integrasi

CSS diintegrasikan ke HTML dengan tiga cara. Cara terbaik yang paling direkomendasikan adalah External CSS — file CSS terpisah yang dihubungkan melalui tag <link> di dalam <head>.

💡 Analogi
Bayangkan HTML adalah kerangka rumah. CSS adalah desain interior — cat warna dinding, jenis lantai, tata letak furnitur. External CSS seperti buku panduan desain yang bisa dipakai untuk banyak halaman sekaligus.
MetodeCaraRekomendasi
Inline CSSstyle="color:red" langsung di tag❌ Hindari — sulit di-maintain
Internal CSSTag <style> di dalam <head>⚠️ Oke untuk halaman kecil
External CSSFile .css terpisah via <link>✅ Best practice industri
  Demo 1 — Integrasi External CSS 
<!-- Di dalam file HTML -->
<head>
  <link rel="stylesheet"
        href="style.css">
</head>
<body>
  <h1>Judul Halaman</h1>
  <p>Selamat datang!</p>
</body>

<!-- Inline (langsung di tag) -->
<h1 style="color: aqua;">
  Judul Inline CSS
</h1>
/* Di dalam file style.css */
h1 {
  color: royalblue;
  font-size: 2em;
  font-family: 'Inter', sans-serif;
}

p {
  color: #374151;
  font-size: 16px;
  line-height: 1.7;
}

✅ Tips
Letakkan tag <link> di dalam <head>, bukan di dalam <body>. Ini memastikan CSS sudah dimuat sebelum konten halaman tampil, mencegah "flash of unstyled content" (FOUC).

02

CSS Selectors

Selector menentukan elemen mana yang akan diberi gaya. Tiga jenis selector dasar yang wajib dikuasai adalah Tag, Class, dan ID selector.

💡 Analogi
Selector seperti cara memanggil orang. Tag selector = panggil semua "dokter". Class selector = panggil semua yang punya topi merah. ID selector = panggil satu orang unik dengan nama lengkapnya.
SelectorSintaks CSSTargetKapan Dipakai
Tagh2 { }Semua tag <h2>Gaya global untuk semua elemen sejenis
Class.artikel { }Semua elemen dengan class="artikel"Kelompok elemen bergaya sama
ID#hero { }Satu elemen dengan id="hero"Satu elemen unik saja
Attributeinput[type="email"] { }Input bertipe emailTargeting berdasarkan atribut
  Demo 2 — CSS Selectors 
<!-- Tag Selector -->
<h2>Ini Tag Selector</h2>

<!-- Class Selector -->
<p class="highlight">
  Ini Class Selector
</p>

<!-- ID Selector -->
<p id="spesial">
  Ini ID Selector
</p>

<!-- Attribute Selector -->
<input type="email"
       placeholder="test@mail.com">
/* Tag Selector */
h2 { color: royalblue; }

/* Class Selector */
.highlight { color: crimson; }

/* ID Selector */
#spesial {
  color: goldenrod;
  font-weight: bold;
}

/* Attribute Selector */
input[type="email"] {
  border: 2px solid royalblue;
  border-radius: 4px;
  padding: 4px 8px;
}

⚠️ Common Mistake
Jangan terlalu banyak memakai ID selector untuk styling! ID bersifat unik dan spesifisitasnya sangat tinggi, membuatnya sulit di-override. Gunakan class untuk styling, simpan ID untuk JavaScript atau anchor link.

03

CSS Box Model & Tipografi

Setiap elemen HTML diperlakukan sebagai sebuah kotak (box). Memahami anatomi kotak ini adalah kunci untuk mengontrol tata letak dan jarak antar elemen.

  • Content — area isi teks/gambar, dikontrol dengan width & height
  • Padding — jarak dalam antara content dan border
  • Border — garis tepi di sekeliling padding
  • Margin — jarak luar antara elemen dengan elemen lain
💡 Analogi
Bayangkan sebuah bingkai foto. Foto = Content. Busa dalam bingkai = Padding. Kayu bingkai = Border. Jarak antar bingkai di dinding = Margin.
  Demo 3 — CSS Box Model 
<!-- Elemen dengan class .kotak -->
<div class="kotak">
  Isi Kotak
</div>

<!-- Elemen paragraf biasa -->
<p class="teks-artikel">
  Teks dengan tipografi custom.
</p>
.kotak {
  width: 200px;
  height: 60px;
  padding: 10px;
  border: 5px solid #2D6BE4;
  border-radius: 20px;
  margin: auto; /* Tengah horizontal */
  background-color: #EBF1FD;
}

/* Tipografi */
.teks-artikel {
  font-family: 'Inter', sans-serif;
  font-size: 16px;
  line-height: 1.7;
  font-weight: 500;
  letter-spacing: 0.02em;
}

04

Properti Latar Belakang (Background)

CSS memungkinkan pengaturan background yang kompleks — dari warna solid, gradient, hingga gambar. Properti-properti ini sangat sering dipakai untuk membuat hero section, jumbotron, atau banner.

PropertiFungsiNilai Umum
background-colorWarna latar belakang solid#hex, rgb(), nama-warna
background-imageGambar sebagai backgroundurl('file.jpg')
background-sizeUkuran gambar backgroundcover, contain, 100%
background-repeatApakah gambar diulangno-repeat, repeat-x
background-attachmentApakah gambar ikut scrollfixed (parallax), scroll
background-positionPosisi gambarcenter, top left
  Demo 4 — Background Properties 
<!-- Jumbotron / Hero Section -->
<div class="jumbotron">
  <h1>Selamat Datang!</h1>
</div>

<!-- Gradient box -->
<div class="gradient-box">
  Gradient Background
</div>
.jumbotron {
  width: 100%;
  height: 200px;
  background-image: url('gambar.jpg');
  background-size: cover;
  background-repeat: no-repeat;
  background-position: center;
}

/* Gradient sebagai background */
.gradient-box {
  background: linear-gradient(
    135deg,
    #2D6BE4,
    #9B59B6
  );
  color: white;
}

/* Efek Parallax saat scroll */
.parallax {
  background-attachment: fixed;
}

✅ Tips — cover vs contain
cover: gambar diperbesar hingga menutupi seluruh area (bisa terpotong). contain: gambar diperkecil hingga seluruh gambar terlihat penuh. Untuk hero section, biasanya pakai cover.
05

Pembuatan Form & Box-Sizing

Menata form agar rapi membutuhkan dua konsep kunci:

  • display: block — membuat input turun ke baris baru secara otomatis
  • box-sizing: border-box — ukuran total elemen sudah termasuk padding & border, tidak "melar"
💡 Analogi — box-sizing
Tanpa border-box: kamu memesan kamar 3x3 meter, tapi dinding dan busa dihitung di luar 3 meter itu. Dengan border-box: 3x3 meter sudah termasuk segalanya — presisi sesuai pesanan.
  Demo 5 — Form Styling & Box-Sizing 
<form action="">

  <label for="user-email">
    Email:
  </label>
  <input type="email"
         id="user-email"
         name="email">

  <label for="pass">
    Password:
  </label>
  <input type="password"
         id="pass"
         name="password">

  <button type="submit">
    Masuk
  </button>

</form>
/* Reset global yang baik */
* { box-sizing: border-box; }

input, textarea {
  display: block;
  width: 100%;
  padding: 10px 14px;
  border: 1px solid #CBD5E1;
  border-radius: 8px;
  font-size: 14px;
  margin-bottom: 12px;
}

/* Attribute selector */
input[type="email"] {
  border-color: #2D6BE4;
}

input:focus {
  outline: none;
  border-color: #2D6BE4;
  box-shadow: 0 0 0 3px rgba(45,107,228,.2);
}

06

Aksesibilitas & Interaksi (:hover & :focus)

Dua konsep penting dalam membuat form yang aksesibel dan interaktif:

  • Tag <label> dihubungkan ke <input> via atribut for ↔ id — klik label = kursor masuk ke input
  • Pseudo-class :hover — mengubah gaya saat kursor melintas di atas elemen
  • Pseudo-class :focus — mengubah gaya saat elemen sedang aktif/diklik
  Demo 6 — Label, :hover & :focus 
<!-- Label terhubung ke input -->
<label for="email">
  Email:
</label>
<input type="email"
       id="email"
       name="email">

<button type="submit">
  Kirim
</button>
/* Pseudo-class :hover */
button {
  background: #2D6BE4;
  color: white;
  border: none;
  padding: 8px 20px;
  border-radius: 6px;
  transition: all 0.2s ease;
}

button:hover {
  background: #1A4DB5;
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(0,0,0,.2);
}

/* Pseudo-class :focus */
input:focus {
  outline: none;
  border-color: #2D6BE4;
  box-shadow: 0 0 0 3px #EBF1FD;
}

✅ Tips — transition untuk animasi halus
Tambahkan transition: all 0.2s ease; pada elemen yang punya :hover agar perubahan gaya berlangsung halus dan smooth. Ini membuat UI terasa jauh lebih premium!
07

Pembuatan Menu Navigasi Vertikal

Mengubah <ul> menjadi menu navigasi yang rapi membutuhkan beberapa teknik: reset bullet, ubah <a> menjadi block, dan selaraskan teks secara vertikal.

💡 Analogi
<a> defaultnya seperti kata dalam kalimat — hanya selebar teksnya. Dengan display: block, ia jadi seperti baris penuh — seluruh area baris bisa diklik, bukan hanya teksnya saja.
  Demo 7 — Navigasi Vertikal 
<!-- Struktur Navigasi Vertikal -->
<nav>
  <ul class="nav-vertikal">

    <li>
      <a href="#">
        🏠 Beranda
      </a>
    </li>

    <li>
      <a href="#">
        📚 Belajar HTML
      </a>
    </li>

    <li>
      <a href="#">
        🎨 Belajar CSS
      </a>
    </li>

  </ul>
</nav>
.nav-vertikal {
  list-style: none; /* Hapus bullet */
  padding: 0;
  margin: 0;
  width: 200px;
}

.nav-vertikal li a {
  display: block; /* Area klik penuh */
  text-decoration: none;
  background-color: #2D6BE4;
  color: white;
  height: 40px;
  line-height: 40px; /* Tengah vertikal */
  padding-left: 16px;
  border-left: 3px solid transparent;
  transition: all 0.15s;
}

.nav-vertikal li a:hover {
  background-color: goldenrod;
  color: black;
  padding-left: 22px;
}

08

Navigasi Horizontal (Navbar)

Untuk membuat menu berjejer mendatar seperti navbar, kita menggunakan teknik float: left pada elemen link sehingga setiap item bergeser rapat ke sisi kiri item sebelumnya.

  Demo 8 — Navigasi Horizontal dengan Float 
<!-- Struktur Navbar Horizontal -->
<nav>
  <ul class="navbar">

    <li>
      <a href="#">
        Beranda
      </a>
    </li>

    <li>
      <a href="#">
        Kelas Web
      </a>
    </li>

    <li>
      <a href="#">
        Kontak
      </a>
    </li>

  </ul>
</nav>
.navbar {
  list-style: none;
  padding: 0;
  margin: 0;
  background-color: #1A1D2E;
  overflow: hidden; /* Wrap float children */
}

.navbar li { float: left; }

.navbar li a {
  display: block;
  color: white;
  text-decoration: none;
  padding: 0 20px;
  height: 44px;
  line-height: 44px;
  border-right: 1px solid rgba(255,255,255,0.1);
  transition: all 0.15s;
}

.navbar li a:hover {
  background-color: #2D6BE4;
}

✅ Tips — overflow: hidden pada parent ul
Karena elemen yang di-float keluar dari aliran layout normal, parent <ul> tidak bisa mendeteksi tingginya. Tambahkan overflow: hidden agar parent menyesuaikan tingginya (teknik clearfix).
09

Tata Letak Konten: Float & Clear

Properti float sangat berguna untuk layout gaya majalah — teks mengalir membungkus di samping gambar. Namun karena elemen float "melayang" di luar aliran normal, kita perlu clear untuk mengembalikan layout ke kondisi normal.

💡 Analogi
Elemen yang di-float seperti es batu di gelas air — es mengapung, dan air (teks) mengalir di sekitarnya. clear seperti menaruh sebuah bata pemisah — semua elemen setelahnya dimulai dari bawah.
  Demo 9 — Float & Clear Layout 
<!-- Artikel bergaya majalah -->
<article class="artikel">

  <img src="foto.jpg"
       class="foto-profil"
       alt="Foto Profil">

  <p>
    Teks artikel yang panjang
    mengalir di samping gambar...
  </p>

  <!-- Wajib ada untuk clear float! -->
  <div class="clear"></div>

</article>
<hr>
.foto-profil {
  width: 100px;
  height: 100px;
  float: left; /* Teks alir di kanan */
  margin: 0 16px 8px 0;
  border-radius: 100%; /* Lingkaran */
  object-fit: cover;
}

.clear {
  clear: both; /* Hentikan efek float */
}

/* Clearfix modern (alternatif) */
.artikel::after {
  content: "";
  display: table;
  clear: both;
}

⚠️ Perhatian — Float vs Flexbox/Grid
Di dunia web modern, Flexbox dan CSS Grid sudah menggantikan teknik float untuk layout. Float sebaiknya hanya dipakai untuk text wrapping gambar. Untuk navbar dan grid konten, gunakan Flexbox.

</article>

  border-radius: 100%;
}
